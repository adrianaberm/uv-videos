### Installation

Clone the repository.   

In the project directory, you can run:

```bash
yarn install
```

Run the app in the development mode:

```bash
yarn dev
```
 
Open [http://localhost:3000](http://localhost:300) to view it in the browser.

```bash
yarn build
```
Builds the app for production.
